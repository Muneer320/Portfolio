# RehabFlow AI — AI-Powered Rehabilitation Planning

An AI-assisted home rehabilitation planner for musculoskeletal conditions, using MedGemma and BLIP for clinical image analysis and personalized recovery plans.

## Tech Stack
Next.js 15, React, FastAPI, Supabase, PostgreSQL, Redis, Modal, MedGemma-4B, BLIP, Docker, Tailwind CSS, TypeScript

## Features
- 5-step guided injury assessment with image upload and pain mapping
- AI-powered analysis: BLIP captions images, MedGemma-4B generates rehab plans
- Personalized dashboard with health metrics and progress tracking
- Day-by-day structured rehab plans with phases and videos
- 10+ language support (English, Hindi, Arabic, French, German, Japanese, etc.)
- YouTube integration with curated exercise videos
- Gamification: points, streaks, achievements
