# BOOP — Book Of Organized Puzzles

An automated puzzle book generator — evolved from a Python CLI into a full-stack web app — that creates categorized word search puzzle books with custom word lists, difficulty levels, and instant PDF download.

## Tech Stack
Python, FastAPI, React, ReportLab, Pillow, Docker, JavaScript, CSS, Axios, React Router

## Features
- Topic-wise categorization of puzzles
- 3 difficulty modes: Normal (13x13), Hard (17x17), Bonus (special patterns)
- Auto-generated title page, index, puzzle pages, and solution pages
- Web version with visual config, file uploads, live PDF preview
- Dockerized backend deployment on Hugging Face
- Most popular project: 11 stars, 4 forks on GitHub
