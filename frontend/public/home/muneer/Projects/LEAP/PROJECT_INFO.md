# LEAP — Linked Enigmas And Puzzles

A Sudoku book generator with a twist: each puzzle is cryptographically linked to the solution of the previous one. You can't skip ahead — you have to solve them in order.

## Tech Stack
Python, ReportLab, Pillow

## Features
- 4 difficulty modes: Easy (40 hints), Medium (36 hints), Advanced (27 hints), Grandmaster (18 hints)
- Automated generation of valid, solvable Sudoku grids with linked placeholders
- Professional PDF output with cover, index, instructions, transitions, solutions
- Fully configurable via CLI arguments
- Clean modular architecture
