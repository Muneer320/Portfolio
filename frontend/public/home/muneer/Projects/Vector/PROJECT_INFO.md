# Vector — AI-Powered NSET Exam Preparation

An intelligent tutoring platform for NSET (Scaler School of Technology Entrance Exam) preparation with mock tests, AI interviews, and personalized learning paths.

## Tech Stack
React 19, Vite, Tailwind CSS 4, Framer Motion, Gemini AI, Three.js, React Router, Node.js, Express.js, MongoDB, JavaScript

## Features
- AI-powered tutoring with Gemini for personalized learning paths
- Full-length NSET mock tests with section-wise practice
- AI mock interviews (technical + behavioral) with real-time feedback
- Advanced analytics dashboard with weak area identification
- Adaptive difficulty based on student skill level
- 3D visuals with Three.js integration
