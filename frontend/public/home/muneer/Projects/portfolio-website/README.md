# Portfolio Website

An interactive Linux desktop portfolio built with React.

## Features
- Hyprland-inspired desktop environment
- Functional terminal emulator
- Custom portfolio commands
- File system simulation
- Window management
- Music player integration
- Image viewer
- PDF support

## Technologies
- React.js
- CSS3 with backdrop filters
- JavaScript ES6+
- HTML5 Canvas

## Commands
Try these terminal commands:
- `bio` - Personal background
- `skills` - Technical skills
- `experience` - Work experience
- `projects` - Portfolio projects
- `contact` - Contact information
- `education` - Educational background

## Usage
Explore the desktop, open terminal, and try commands like `bio`, `skills`, `projects`!

## Installation
```bash
npm install
npm start
```
