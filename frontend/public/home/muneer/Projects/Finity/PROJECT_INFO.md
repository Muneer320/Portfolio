# Finity — Gamified Financial Learning & Investment Platform

A hackathon-winning project that democratizes financial literacy through AI-powered education, risk-free trading simulation, and behavioral gamification.

## Tech Stack
React 18, Vite, FastAPI, SQLAlchemy, OpenAI API, Gemini AI, Tailwind CSS 3, Framer Motion, JWT, SQLite

## Features
- AI Financial Coach with personalized, context-aware advice
- Paper trading simulator with 100,000 F-Coins virtual currency
- Investment simulator with risk profiles (Low/Medium/High)
- AI micro-courses generated based on user activity
- Gamification: streaks, achievements (15+), behavioral reinforcement
- Expense & income tracking with visualizations
- Smart onboarding questionnaire for financial profiling
