# BOOP-web — Web Puzzle Book Generator

The full-stack web evolution of BOOP, making automated puzzle book creation accessible through a browser interface.

## Tech Stack
React, FastAPI, Python, Docker, Axios, React Router, ReportLab, Pillow

## Features
- Visual puzzle configuration (difficulty, bonus modes, grid sizes)
- Upload custom word lists, cover images, and backgrounds
- Live PDF preview and one-click download
- FastAPI backend deployed on Hugging Face
- React frontend deployed on Vercel
