# Ascent Dashboard

The central platform for End-to-End hackathon participants management and control — a full-stack event management system powering user onboarding, event participation, team submissions, QR-based validation, coins/sidequest gamification, real-time leaderboards, and an AI-powered Mirage photo gallery.

## Tech Stack
FastAPI, React 18, Python 3.11+, PostgreSQL, Supabase, Redis, SQLAlchemy async, Docker, JWT

## Features
- Event lifecycle management with phase control
- QR-based venue validation with role-based scanner auth
- Gamification: coins, sidequests, rewards, redemption booths
- AI-powered photo gallery (Mirage) with RAW/TIFF/HEIF processing
- Real-time leaderboards with Redis caching
- Admin panel with 24 tabs for full system management
- 32 backend services, 23 route modules, 10 database models
- 41 incremental SQL migrations
