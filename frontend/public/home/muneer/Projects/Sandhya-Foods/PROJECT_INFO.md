# Sandhya Foods — E-Commerce Platform

A modern e-commerce website for Sandhya Food Products with Razorpay payment integration, Supabase backend, and WhatsApp order notifications.

## Tech Stack
Next.js 16, React 19, TypeScript, Supabase, PostgreSQL, Razorpay, TanStack Query, Framer Motion, Zustand, Tailwind CSS 4, Recharts

## Features
- Product catalog with detailed descriptions and images
- Shopping cart with real-time price calculation
- Razorpay payment integration with webhook handling
- WhatsApp order notifications
- Admin dashboard with Supabase Row-Level Security
- Responsive design with Framer Motion animations
- Analytics-ready with Recharts integration
