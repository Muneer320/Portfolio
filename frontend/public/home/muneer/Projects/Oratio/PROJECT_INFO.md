# Oratio — AI-Powered Debate Arena

A voice-enabled, AI-powered debate platform where hosts create rooms, participants debate via text or voice, spectators reward debaters in real-time, and an AI judge scores rounds using the LCR Model (Logic, Credibility, Rhetoric).

## Tech Stack
FastAPI, React 18, Python, Gemini AI, WebSockets, Socket.IO, SQLAlchemy, Docker, Docker Compose, Nginx, Tailwind CSS, Framer Motion, JWT

## Features
- Real-time debate rooms with text/voice participation
- Spectator rewards with real-time emoji reactions
- AI Judge using LCR (Logic, Credibility, Rhetoric) scoring
- Multiple AI fallback tiers (Gemini → Replit AI → Static)
- 10 backend route modules, 15 frontend pages
- Dockerized deployment with Nginx reverse proxy
