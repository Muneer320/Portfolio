# Surjit Hockey Tournament Website

A full-stack tournament management platform for the annual IndianOil Servo Surjit Hockey Tournament (Grade-I), featuring team management, news, sponsor showcases, and an admin dashboard.

## Tech Stack
FastAPI, React 19, Vite, MySQL, SQLAlchemy, Tailwind CSS 4, Framer Motion, shadcn/ui, JWT

## Features
- Tournament management with editions, teams, match results
- News & announcements with image support
- Team profiles with detailed stats
- Committee directory (4 core + 75+ team members)
- Sponsor & patron showcases
- JWT-authenticated admin CMS dashboard
- API-first with auto-generated OpenAPI docs
