# SST Lounge Bot — Discord Community Bot

A feature-rich Discord bot for SST Batch '29 with contest tracking across Codeforces, CodeChef, AtCoder, and LeetCode, automatic role management, and an auto-update system.

## Tech Stack
Python, discord.py, SQLite, clist.by API, GitHub API

## Features
- Multi-platform contest tracking with real-time status
- Automated daily contest announcements
- Smart filtering by platform, date range, and limits
- Discord Veteran role auto-assignment for 5+ year accounts
- Auto-update system with GitHub release monitoring
- Three-tier admin system (Server Owner → Discord Admin → Bot Admin)
- Advanced logging with file-based export and filters
